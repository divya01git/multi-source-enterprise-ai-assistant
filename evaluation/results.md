## 6. SQL — Top Selling Product

### Question

Which product generated the highest sales revenue?

### Expected Route

```text
SQL
```

### Expected SQL Pattern

```sql
SELECT
    p.product_name,
    SUM(o.quantity * p.price) AS total_sales
FROM orders o
JOIN products p
    ON o.product_id = p.product_id
GROUP BY p.product_name
ORDER BY total_sales DESC
LIMIT 1;
```

### Status

**PASS**

# Known Limitations During Testing

Some tests may fail when the external LLM provider reaches its API rate limit.

For example:

```text

```

This is an external API availability limitation and does not represent a database or SQL execution failure.

When the LLM API is available, the SQL, RAG, Web, and multi-source workflows can be executed through the complete LangGraph pipeline.
